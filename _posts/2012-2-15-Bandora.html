Introduction

Bandora is an attempt to merge Bandcamp with reccomendation radio systems like Pandora or Last.fm. Bandcamp does not provide a way to query tracks based off of relatable things like tags, so most of the project was violating the Bandcamp TOS and abusing thier API. Sorry Bandcamp!

Results

The project is currently in development and hosted here.

Source

The project is based off a Ruby backend and a Javascript frontend, with communication with Websockets. Source for the web app is availble on github here.
