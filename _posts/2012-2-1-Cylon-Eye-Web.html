Introduction

The Cylon Eye is a CSH project which is a series of modular LED displays. The purpose of the web app is to provide a method for people to create animations to be pushed to the actual displays. The app is based off of Javascript and allows users to enter and simulate potential animations and then send them to a server to be parsed and sent on to the LED panels.
